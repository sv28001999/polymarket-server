# Node.js REST API Project

A RESTful API built with Node.js and Express.

## Features

- RESTful endpoints
- Server time integration with Polymarket
- Error handling
- Clean code structure

## Installation

1. Clone the repository:
```bash
git clone <your-repository-url>
cd <project-folder>
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file:
```bash
PORT=3000
NODE_ENV=development
```

4. Run the application:
```bash
# Development mode
npm run dev

# Production mode
npm start
```
